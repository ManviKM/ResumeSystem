Resume Match Analyzer

A Java-based Resume Match Analyzer that compares a candidate’s resume skills against a predefined set of job-required skills and calculates the skill match percentage. This project helps evaluate how well a candidate’s skill set aligns with job requirements.

Features

* Uses predefined job-required skills.
* Accepts resume skills as user input.
* Performs case-insensitive skill matching.
* Counts matching skills between the resume and job requirements.
* Calculates and displays the overall match percentage.
* Provides a simple way to assess candidate-job compatibility.

Technologies Used

* Java
* Scanner Class
* Arrays
* String Manipulation

How It Works

1. The program stores the required job skills.
2. The user enters skills listed in their resume.
3. The program compares the resume skills with the required skills.
4. It calculates:
    * Number of matched skills
    * Skill match percentage
5. The results are displayed to the user.
